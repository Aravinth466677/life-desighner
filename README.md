# 🏡 Life Designer – Interior Design Website

A responsive web application for an interior design service that allows users to explore projects and enables admins to manage content dynamically.

This project focuses on building a clean UI, secure authentication, and real-world content management features.

---

## 🚀 Features

- Responsive website for showcasing interior design projects
- Admin authentication using JWT
- Admin dashboard to:
  - Add new projects
  - Update existing content
  - Manage project details dynamically
- Secure access control for admin-only features
- Clean and user-friendly UI

---

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6+)
- HTML5 & CSS3
- Tailwind CSS / Bootstrap
- JWT Authentication
- REST API (for data handling)
